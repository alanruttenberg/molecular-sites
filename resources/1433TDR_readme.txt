1433TDR Readme file

The columns in the file are as follow:

UniprotId	UniProtKB ID
EntryDescription	name in TDR repository
ProteoformId	ID in TDR repository
AccessionNumber	UniProtKB Acc, it includes isoform accessions when applicable
Modifications	the first number indicate position in the protein sequence deposited in TDR and not in the sequence reference in AccessionNumber, where 0 is the first aminoacid in the sequence. THis if followed by the type of modification using MOD ID), if more than one modification exists in same form these are separated by pipes. E.g.0, MOD:00058|1, MOD:00060
IsoformDescription	DE lines from UniProtKB
